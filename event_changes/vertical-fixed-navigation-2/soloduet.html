Rules

  • This module consists of two sub modules i.e Solo and Duet, judged separately, conducted in the same time slot.

  • Any dance style and song is allowed. Tracks are to be submitted 5 days before the event.

  • Both modules will have just one round
  • Props can be used if required, but to allow it is on the discretion of organizers.
  • List of props is to be submitted 5 days before the event or else the prop shall not be allowed
  • Registration fees: Duet - Rs. 200; Solo - Rs. 100

  • Team size: Solo- 1, Duet-2

  • Time duration : 4-6 min(+1 min preparation time)
